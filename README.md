# NickColorPlugin 🎨

Современный плагин для Minecraft (Paper 1.21.1+), позволяющий игрокам менять цвет своего ника на HEX-цвета и красивые градиенты.

## Особенности
- **Полная поддержка MiniMessage:** Продвинутое форматирование текста без устаревших `&` кодов.
- **Поддержка стандартных цветов:** Можно использовать названия ванильных цветов (например, `red`, `blue`, `gold`) наравне с HEX-кодами.
- **Интеграция с FlectonePulse (PlaceholderAPI):** Плагин выступает как провайдер данных. Чат и TabList контролируются вашим глобальным плагином на форматы, а NickColorPlugin передает ему цвет.
- **Градиенты над головой (NameTag):** Использует современные `TextDisplay` сущности для отрисовки красивых, полностью кастомных ников над головой персонажа. Поддерживает ванильное поведение приседания (сдвиг вниз и полупрозрачность).
- **SQLite база данных:** Цвета асинхронно сохраняются в локальную базу данных `database.db`.
- **Пресеты:** Удобное кликабельное меню готовых градиентов для игроков.
- **Полностью на русском языке:** Никакого английского в конфигах или сообщениях.

## Команды

Основная команда: `/nickcolor` (Алиасы: `/nc`, `/color`, `/nickc`)

### Для игроков
- `/nc set <#HEX | цвет>` — Установить сплошной цвет (например, `/nc set #FF5555` или `/nc set red`).
- `/nc set <цвет1>:<цвет2>:[цвет3]...` — Установить градиент из двух и более цветов (например, `/nc set #FF0000:#00FF00:#0000FF`).
- `/nc set random` — Выбрать случайный сплошной цвет.
- `/nc set random gradient` — Выбрать случайный градиент.
- `/nc presets` — Открыть интерактивное меню выбора готовых пресетов.
- `/nc reset` — Сбросить цвет ника на стандартный.
- `/nc help` — Показать справку по командам.

### Для администраторов
- `/nc admin set <игрок> <цвет|c1:c2:c3...|random> [gradient]` — Установить цвет, градиент или случайный цвет другому игроку.
- `/nc admin reset <игрок>` — Сбросить цвет другому игроку.
- `/nc admin reload` — Перезагрузить конфигурационный файл.

## Права (Permissions)
- `nickcolor.use` — Доступ к базовым командам для изменения своего цвета (по умолчанию: `true`).
- `nickcolor.admin.set` — Доступ к изменению цвета других игроков (по умолчанию: `op`).
- `nickcolor.admin.reload` — Доступ к команде перезагрузки плагина (по умолчанию: `op`).

## Интеграция с FlectonePulse

Так как `NickColorPlugin` делегирует форматирование чата и Tab-листа плагину **FlectonePulse**, вам нужно правильно настроить плейсхолдеры.

1. Убедитесь, что у вас установлен **PlaceholderAPI**.
2. В конфигурации `FlectonePulse` (или любого другого плагина на форматы), используйте следующие плейсхолдеры в форматах чата и таба:
   - `%nickcolor_tag%` — Возвращает открывающий тег цвета (например, `<#FF5555>`).
   - `%nickcolor_name%` — Возвращает полностью готовый, закрытый тегами ник игрока (например, `<gradient:#FF0000:#00FF00>Notch</gradient>`).

**Важно:** Обязательно ВЫКЛЮЧИТЕ модуль `scoreboard`/`nametag` в самом FlectonePulse, так как NickColorPlugin берет ответственность за отрисовку ников над головой на себя, используя TextDisplay.

## Технические детали
Плагин требует Java 21+ и сервер на базе PaperMC (версии 1.21.1+). Для сборки используйте Maven: `mvn clean package`.
